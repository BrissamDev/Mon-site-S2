A propos de moi

Présentation personnelle

Étudiant en droit privé et sciences criminelles passionné par les technologies numériques et le développement web.
Mon objectif est de combiner droit et digital pour construire un profil polyvalent.

Compétences

Mon profil s'articule autour de deux axes principaux :

Axe juridique

  • Analyse juridique
  • Droit pénal et les sciences criminelles
  • Risonnement juridique

Axe numérique

Je développe également :

  • HTML - CSS - VS Code - bases du Développement web
  • l'utilisation de l'éditeur VS Code
  • les compétences en bureautique
  • la familiarité avec les environnements informatiques et les réseaux

Vision

Mon objectif est de construire un profil polyvalent, capable d'évoluer dans le domaine juridique tout en maîtrisant les outils numériques.

Je souhaite devenir un professionnel capable de comprendre les enjeux du droit dans un monde de plus en plus digitalisé, et d'utiliser la technologie comme un outil d'efficacité, d'analyse et de création.

Centre d'interêt

Actuellement en apprentissage du développement web Fullstack à Akieni Academy, je poursuis également le développement de mes compétences dans les domaines du droit, du numérique et des technologies de l'information.
